## 为什么要花钱买邮箱？

这问题我被问过很多次。"Gmail免费啊，为什么要付费？"

免费的东西，往往你不是用户，你是产品。Gmail确实不收你钱，但它在用你的数据赚钱。每封邮件都被扫描，每个关键词都被记录，然后精准投放广告。这就是"免费"的代价。

TheXYZ走的是另一条路：你付钱，我提供服务，两清。不扫描，不追踪，不卖数据。商业模式很简单，但在这个时代反而显得特别珍贵。

根据多个评测网站的用户反馈（GetApp、TechRadar、Capterra等），TheXYZ的服务获得了相当高的评价。用户普遍认可的几个点：

**1. 垃圾邮件过滤真的强**  
有用户说自己从Gmail迁移过来，最大的惊喜是垃圾邮件几乎绝迹了。不是完全没有，而是过滤准确率极高。不像某些邮箱，把正常邮件当垃圾，或者让垃圾邮件大摇大摆进收件箱。

不过也有少数用户反映，TheXYZ的垃圾邮件过滤过于严格，有时连Amazon、PayPal的邮件都会被误判。这种情况下需要手动添加白名单。

**2. 无限别名功能太好用**  
这个功能解决了一个大问题：你不用到处留真实邮箱了。

比如你在某个网站注册账号，可以创建一个临时别名，比如 `randomsite@yourdomain.com`。如果这个网站泄露数据或者卖邮箱，你直接删掉这个别名就行，不影响主邮箱。有用户说自己给每个网站都分配独立别名，谁卖了数据一查就知道。

从Gmail或其他邮箱搬家，很多人担心数据会丢。TheXYZ提供付费迁移服务（$9.95一次性收费），据用户反馈，几十万封邮件都能完整迁移过来，没什么技术门槛。
